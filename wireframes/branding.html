Branding & Tampilan

Atur tampilan aplikasi untuk dirimu sendiri — logo, favicon, semua token warna, & Custom CSS. Default-nya tema bawaan "Coral Sunset". Ubah sesukamu, lalu Simpan; bisa Reset kapan saja.

Akun:
i
Perubahan hanya berlaku untuk akun kamu. Rekan satu tim & tenant tetap melihat tampilan mereka sendiri. Branding ini disimpan per-user (bukan se-tenant), diterapkan saat sesi login kamu.

3 · Advanced — Custom CSS

Override penuh lewat CSS mentah (override :root & selector). Untuk pengguna lanjutan.

/* :root { --primary: 12 96% 67%; } … textarea Custom CSS */

Disuntik setelah token di atas, jadi menang terhadap skema warna. Hanya untuk sesi kamu.
